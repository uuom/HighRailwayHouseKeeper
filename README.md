# HighRailwayHouseKeeper
React-Native 实现的高铁管家

####项目说明：
    和[信部落](https://github.com/uuom/aitribe)一样，只完成简单UI页面展示，具体页面逻辑没有添加。
    项目中引用了redux，但是由于没添加数据和业务逻辑，并没有使用。
